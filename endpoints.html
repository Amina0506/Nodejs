API Endpoints

Books

HTTP Endpoint Beschrijving Opties / Params
GET /api/books Alle boeken ophalen ?limit=<nummer>&offset=<nummer>
GET /api/books/:id 1 boek ophalen /
POST /api/books Nieuw boek toevoegen Velden: { title, author, year, genre, content }
PUT /api/books/:id Boek updaten Velden updaten naar keuze: { title, author, year, genre, content }
DELETE /api/books/:id Boek verwijderen /
GET /api/books-search Boeken zoeken op titel of auteur ?query=<zoekterm>

Authors

HTTP Endpoint Beschrijving Opties / Params
GET /api/authors Alle auteurs ophalen ?limit=<nummer>&offset=<nummer>&sortBy=name/birthYear
GET /api/authors/:id 1 auteur ophalen /
POST /api/authors Nieuwe auteur toevoegen Velden: { name, birthYear, bio, genres }
PUT /api/authors/:id Auteur updaten Velden updaten naar keuze: { name, birthYear, bio, genres }
DELETE /api/authors/:id Auteur verwijderen /
GET /api/authors-search Auteurs zoeken op naam ?query=<zoekterm>
